# QR Generator App 🛡️

Proyecto de desarrollo seguro con IA para el IES Rafael Alberti.

## 🚀 Características
- Generación de códigos QR mediante `qrcode`.
- **Seguridad**: Validación estricta de inputs y límite de 2000 caracteres.
- **Testing**: Cobertura de Happy Path y casos de error con Mocha/Chai.

## 🛠️ Instalación y Uso
1. `npm install`
2. `npm start` (Servidor en puerto 3000)
3. `npm test` (Para ejecutar la suite de pruebas)

## 🤖 Uso de IA
- Handler generado con **Gemini**.
- Revisión manual línea a línea para asegurar el manejo de errores y tipos de datos.
